Introduction

In programming, data type is an important concept. Variables can store data of different types, and different types can do different things.

Data types are an essential concept in the Python programming language. In Python, every value has its own Python data type. The classification of data items or to put the data value into some sort of data category is called Data Types. It helps to understand what kind of operations can be performed on a value.

We can divide these data types in the following groups:

  • Text type (String);
  • Numeric types (Int, Float, Complex);
  • Sequence types (List, Tuple, Range);
  • Mapping type (Dictionary);
  • Set types (Set, Frozen Set);
  • Boolean Type (Bool);
  • Binary Types (Bytes, Bytearray, Memoryview)

Let's take a look at each Python data type in the next sections.

String

Strings in Python are surrounded by either single quotation marks, or double quotation marks.

a = "Hello" print(a)

Int

Int, or integer, is a whole number, positive or negative, without decimals, of unlimited length.

a = 12345 print(a)

Float

Float, or "floating point number" is a number, positive or negative, containing one or more decimals.

a = 3.1415 print(a)

Complex

Complex numbers are written with a "j" as the imaginary part.

a = 3 + 4j print(a)

List

Lists are used to store multiple items in a single variable.

Lists are created using square brackets.

a = [1, 2, 3, 5, 8, 13, 21] print(a)

Tuple

Tuples are used to store multiple items in a single variable.

A tuple is a collection which is ordered and unchangeable. That is saying:

  • When we say that tuples are ordered, it means that the items have a defined order, and that order will not change;
  • Tuples are unchangeable, meaning that we cannot change, add or remove items after the tuple has been created.

Tuples are written with round brackets.

a = (0,0,1) print(a)

Range

The Python range type generates a sequence of integers by defining a start and the end point of the range. It is generally used with the for loop to iterate over a sequence of numbers.

a = range(10) print(a)

Dictionary

Dictionaries are used to store data values in key:value pairs.

Dictionaries are written with curly brackets, and have keys and values.

a = {'texas': 'austin', 'california': 'sacramento'} print(a)

Set

Sets are used to store multiple items in a single variable.

A set is a collection which is unordered and unindexed. That is saying:

  • Unordered means that the items in a set do not have a defined order;
  • Set items can appear in a different order every time you use them, and cannot be referred to by index or key.
a = {1, 2, 3} print(a)

Frozen Set

Frozen set is just an immutable version of a Python set object. While elements of a set can be modified at any time, elements of the frozen set remain the same after creation.

Boolean

In programming you often need to know if an expression is True or False.

You can evaluate any expression in Python, and get one of two answers, True or False.

When you compare two values, the expression is evaluated and Python returns the Boolean answer.

Bytes

In short, the bytes type is a sequence of bytes that have been encoded and are ready to be stored in memory/disk.

Byte Array

A bytearray object is an array of given bytes.

Memory View

A memory view is a safe way to expose the buffer protocol in Python.

It allows you to access the internal buffers of an object by creating a memory view object.
